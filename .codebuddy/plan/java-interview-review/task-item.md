# Java 面试复习实施计划

> 按照需求文档中的三梯队优先级排列，建议按顺序逐步推进。

---

## 🥇 第一梯队（必须精通，占 50%+ 时间）

- [ ] 1. 复习 Java 基础知识与 JVM 原理
   - 梳理面向对象四大特性（封装/继承/多态/抽象），整理接口 vs 抽象类的对比笔记
   - 手写 HashMap 扩容流程，总结多线程下使用 HashMap 的风险及替代方案（ConcurrentHashMap）
   - 整理线程池七大参数含义及四种拒绝策略，能口述一次 OOM 问题的排查步骤
   - 绘制 JVM 内存分区图，对比 G1 与 CMS 的区别，理解 AQS 等待队列与 CAS 的 ABA 问题
   - _需求：1.1、1.2、1.3、1.4、1.5、1.6_

- [ ] 2. 复习设计模式（GoF 23 种）
   - 手写单例模式的 5 种实现（饿汉/懒汉/双重检查/静态内部类/枚举），重点理解 volatile 的必要性
   - 用代码示例实现策略模式消除 if-else、模板方法固定骨架、观察者模式发布订阅
   - 整理代理模式（JDK 动态代理 vs CGLIB）与装饰器模式的本质区别
   - 梳理 Spring 框架中至少 5 种设计模式的具体应用位置（工厂/单例/代理/模板方法/观察者）
   - _需求：10.1、10.2、10.3、10.4、10.5、10.6_

- [ ] 3. 复习 Spring / Spring Boot 核心原理
   - 手绘 Bean 生命周期流程图（实例化→属性注入→Aware→BeanPostProcessor→初始化→销毁）
   - 梳理 Spring MVC 请求处理完整链路（DispatcherServlet → HandlerMapping → Controller → 响应）
   - 整理事务不生效的 5 种常见原因（非 public/同类调用/异常被捕获/非 Spring 管理/传播行为错误）
   - 理解三级缓存解决循环依赖的原理，说明为何构造器注入无法解决循环依赖
   - _需求：3.1、3.2、3.3、3.4、3.5_

- [ ] 4. 复习 MySQL 索引、事务与性能优化
   - 绘制 B+ 树结构图，对比聚簇索引与二级索引，整理联合索引最左前缀原则及索引失效场景
   - 梳理 MVCC 实现原理（undo log + Read View），理解 RR 隔离级别下如何解决幻读
   - 整理间隙锁/临键锁的加锁范围，理解死锁产生条件及 InnoDB 死锁检测机制
   - 练习用 EXPLAIN 分析执行计划，能识别 type/key/rows 字段含义并给出优化方案
   - _需求：5.1、5.2、5.3、5.4、5.5_

---

## 🥈 第二梯队（需要深入，占 35% 时间）

- [ ] 5. 复习 Java 8 及高版本新特性
   - 练习 Lambda + Stream + Optional 组合使用，将一段传统 for 循环代码重构为函数式风格
   - 整理四大函数式接口（Function/Consumer/Supplier/Predicate）及方法引用四种形式
   - 了解 Java 10-17 关键特性：var 类型推断、文本块、Record 类、密封类与模式匹配 instanceof
   - _需求：2.1、2.2、2.3、2.4、2.5、2.6、2.7_

- [ ] 6. 复习 Redis 缓存设计与高可用
   - 整理五种数据结构的底层编码及适用场景（String/Hash/List/Set/ZSet），重点掌握 ZSet 的 skiplist
   - 梳理缓存穿透/击穿/雪崩的区别与解决方案，能口述布隆过滤器的工作原理
   - 对比 RDB 与 AOF 持久化的数据安全性与性能影响，了解混合持久化方案
   - 整理 Redisson 分布式锁相比 SETNX 的优势（看门狗续期/可重入/红锁）
   - _需求：4.1、4.2、4.3、4.4、4.5_

- [ ] 7. 复习 Kafka 消息队列原理
   - 梳理 Topic/Partition/Replica/Consumer Group/Offset 的含义，理解 Partition 是并行度基本单位
   - 从生产者（acks）、Broker（ISR）、消费者（手动提交 offset）三维度整理消息不丢失方案
   - 整理 Rebalance 触发条件与三种分区分配策略（Range/RoundRobin/Sticky）的区别
   - 理解 Kafka 高吞吐的四个核心机制：顺序写磁盘、零拷贝、批量发送、消息压缩
   - _需求：6.1、6.2、6.3、6.4、6.5_

- [ ] 8. 复习软件工程核心知识
   - 用代码示例说明 SOLID 五条原则，能识别违反各原则的典型代码坏味道
   - 梳理微服务架构核心组件（网关/注册发现/熔断/链路追踪），能说明微服务 vs 单体的取舍
   - 理解 DDD 核心概念（聚合根/值对象/领域事件），对比贫血模型与充血模型的优缺点
   - 整理 CAP 理论，能说明 MySQL（CP）、Redis Cluster（AP）、ZooKeeper（CP）的选择依据
   - _需求：9.1、9.2、9.3、9.4、9.5、9.6_

## 🥉 第三梯队（了解原理，占 15% 时间）

- [ ] 9. 复习 Elasticsearch 搜索引擎
   - 理解倒排索引的构建过程（分词→词项→文档列表），能解释为何比 LIKE 快几个数量级
   - 掌握 match/term/range/bool 查询的使用，理解 query context 与 filter context 的区别
   - 了解主分片数不可修改的原因，掌握写入优化（bulk/减少 refresh）和查询优化（filter 缓存/避免深度分页）
   - _需求：7.1、7.2、7.3、7.4、7.5_

- [ ] 10. 复习 PostgreSQL 核心特性与选型
   - 整理 PG 相比 MySQL 的核心优势（JSONB/GIN 索引/窗口函数/表继承/严格 SQL 标准）
   - 理解 PG 的 MVCC 通过多版本行存储实现，对比 MySQL 通过 undo log 实现的差异
   - 了解 VACUUM/AUTOVACUUM 的作用，理解表膨胀的原因及监控手段
   - 了解 CTE、物化视图、窗口函数（ROW_NUMBER/RANK/LAG）的使用场景
   - _需求：8.1、8.2、8.3、8.4、8.5_
