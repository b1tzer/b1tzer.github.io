    зҹҘиҜҶзӮ№еҜјиҲӘВ¶

    # зҹҘиҜҶзӮ№ ж ёеҝғдёҖеҸҘиҜқ иҜҰз»Ҷж–ҮжЎЈ
    01 йқўеҗ‘еҜ№иұЎ е°ҒиЈ…йҡҗи—Ҹе®һзҺ°гҖҒз»§жүҝеӨҚз”Ёд»Јз ҒгҖҒеӨҡжҖҒи§ЈиҖҰи°ғз”Ёж–№гҖҒжҠҪиұЎе®ҡд№үи§„иҢғ йқўеҗ‘еҜ№иұЎ
    02 йӣҶеҗҲжЎҶжһ¶ ArrayList йҡҸжңәи®ҝй—®еҝ«гҖҒLinkedList еўһеҲ еҝ«гҖҒHashMap ж•°з»„+й“ҫиЎЁ+зәўй»‘ж ‘гҖҒConcurrentHashMap зәҝзЁӢе®үе…Ё йӣҶеҗҲжЎҶжһ¶
    03 е№¶еҸ‘зј–зЁӢ synchronized дҝқиҜҒеҺҹеӯҗжҖ§+еҸҜи§ҒжҖ§гҖҒvolatile дҝқиҜҒеҸҜи§ҒжҖ§+жңүеәҸжҖ§гҖҒзәҝзЁӢжұ йҒҝе…Қйў‘з№ҒеҲӣе»әй”ҖжҜҒзәҝзЁӢ е№¶еҸ‘зј–зЁӢ
    04 JVM еҶ…еӯҳз»“жһ„дёҺ GC е ҶеӯҳеҜ№иұЎгҖҒж Ҳеӯҳеё§гҖҒе…ғз©әй—ҙеӯҳзұ»дҝЎжҒҜпјӣGC еҲҶд»Јж”¶йӣҶпјҢG1 жҳҜ JDK9+ й»ҳи®Өж”¶йӣҶеҷЁ JVMеҶ…еӯҳз»“жһ„дёҺGC
    05 ејӮеёёеӨ„зҗҶ Checked зј–иҜ‘еҷЁејәеҲ¶еӨ„зҗҶгҖҒUnchecked зј–зЁӢй”ҷиҜҜеә”дҝ®еӨҚд»Јз ҒпјӣзҰҒжӯўз©ә catch еқ— ејӮеёёеӨ„зҗҶ
    06 AQS дёҺ CAS AQS жҳҜе№¶еҸ‘еҢ…ж ёеҝғжЎҶжһ¶пјҢCAS жҳҜж— й”Ғзј–зЁӢеҹәзЎҖпјӣReentrantLock жҜ” synchronized жӣҙзҒөжҙ» AQSдёҺCAS
    07 [Java8] еҮҪж•°ејҸзј–зЁӢ Lambda иЎЁиҫҫејҸ + Stream жөҒејҸзј–зЁӢ + Optional з©әеҖјеӨ„зҗҶпјҢеҮҪж•°ејҸзј–зЁӢдёүеӨ§ж ёеҝғ Java8еҮҪж•°ејҸзј–зЁӢ
    08 [Java8] е…¶д»–ж–°зү№жҖ§ ж–°ж—Ҙжңҹ APIпјҲjava.timeпјү+ жҺҘеҸЈй»ҳи®Өж–№жі•дёҺйқҷжҖҒж–№жі• Java8е…¶д»–ж–°зү№жҖ§
    09 [Java9-17] ж–°зү№жҖ§ var еұҖйғЁеҸҳйҮҸжҺЁж–ӯгҖҒRecordгҖҒSealed ClassesгҖҒPattern Matching зӯү Java9-17ж–°зү№жҖ§
    10 жіЁи§Ј е…ғжіЁи§Је®ҡд№үжіЁи§ЈиЎҢдёәпјҢиҮӘе®ҡд№үжіЁи§Ј + еҸҚе°„/APT е®һзҺ°жЎҶжһ¶еҠҹиғҪ жіЁи§ЈAnnotation
    11 ж•°жҚ®з»“жһ„зІҫи®І зәўй»‘ж ‘иҮӘе№іиЎЎ O(log n)гҖҒB+ж ‘зЈҒзӣҳеҸӢеҘҪгҖҒи·іиЎЁжҰӮзҺҮе№іиЎЎгҖҒеёғйҡҶиҝҮж»ӨеҷЁеҲӨеӯҳеңЁ ж•°жҚ®з»“жһ„зІҫи®І
    12 зұ»еҠ иҪҪжңәеҲ¶дёҺеҸҢдәІе§”жҙҫжЁЎеһӢ Boot вҶ’ Ext/Platform вҶ’ App дёүзә§е§”жҙҫпјӣSPI / Tomcat / OSGi з ҙеқҸе§”жҙҫзҡ„е…ёеһӢеңәжҷҜ зұ»еҠ иҪҪжңәеҲ¶дёҺеҸҢдәІе§”жҙҫжЁЎеһӢ
    13 еӯ—з¬ҰдёІеә•еұӮеҺҹзҗҶдёҺ StringPool еӯ—з¬ҰдёІдёҚеҸҜеҸҳзҡ„жң¬иҙЁгҖҒintern() дёҺиҝҗиЎҢж—¶еёёйҮҸжұ гҖҒJDK 9+ зҙ§еҮ‘еӯ—з¬ҰдёІ еӯ—з¬ҰдёІеә•еұӮеҺҹзҗҶдёҺStringPool
    14 жіӣеһӢеә•еұӮеҺҹзҗҶдёҺзұ»еһӢж“ҰйҷӨ зј–иҜ‘жңҹжЈҖжҹҘ + иҝҗиЎҢжңҹж“ҰйҷӨпјӣжЎҘжҺҘж–№жі•дёҺйҖҡй…Қз¬Ұзҡ„еә•еұӮжңәеҲ¶ жіӣеһӢеә•еұӮеҺҹзҗҶдёҺзұ»еһӢж“ҰйҷӨ
    15 еҸҚе°„жҖ§иғҪдёҺ MethodHandle Method.invoke зҡ„ејҖй”ҖжқҘжәҗгҖҒMethodHandle дёҺ invokedynamic зҡ„дјҳеҠҝ еҸҚе°„дёҺMethodHandle
    16 NIO дёҺ IO жЁЎеһӢж·ұеәҰи§Јжһҗ BIO/NIO/AIO еҜ№жҜ”гҖҒReactor жЁЎеһӢгҖҒepoll дёҺ Netty зҡ„е…ізі» NIOдёҺIOжЁЎеһӢж·ұеәҰи§Јжһҗ

    й«ҳйў‘й—®йўҳзҙўеј•В¶

    й—®йўҳ иҜҰи§Ғ
    йқўеҗ‘еҜ№иұЎеӣӣеӨ§зү№жҖ§еҲҶеҲ«и§ЈеҶід»Җд№Ҳй—®йўҳпјҹ йқўеҗ‘еҜ№иұЎ
    HashMap жү©е®№жөҒзЁӢпјҹJDK7 еӨҙжҸ’жі•дёәд»Җд№Ҳдјҡжӯ»еҫӘзҺҜпјҹ йӣҶеҗҲжЎҶжһ¶
    synchronized е’Ң volatile зҡ„еҢәеҲ«пјҹ е№¶еҸ‘зј–зЁӢ
    зәҝзЁӢжұ ж ёеҝғеҸӮж•°жҖҺд№Ҳи®ҫзҪ®пјҹдёәд»Җд№ҲдёҚз”Ё Executorsпјҹ е№¶еҸ‘зј–зЁӢ
    ThreadLocal дёәд»Җд№ҲдјҡеҶ…еӯҳжі„жјҸпјҹ е№¶еҸ‘зј–зЁӢ
    JVM еҶ…еӯҳеҲҶеҢәжңүе“Әдәӣпјҹеҗ„иҮӘеӯҳд»Җд№Ҳпјҹ JVMеҶ…еӯҳз»“жһ„дёҺGC
    G1 е’Ң CMS зҡ„еҢәеҲ«пјҹ JVMеҶ…еӯҳз»“жһ„дёҺGC
    OOM й—®йўҳеҰӮдҪ•жҺ’жҹҘпјҹ JVMеҶ…еӯҳз»“жһ„дёҺGC
    еҸҢйҮҚжЈҖжҹҘй”Ғзҡ„еҚ•дҫӢдёәд»Җд№ҲйңҖиҰҒ volatileпјҹ AQSдёҺCAS
    AQS зӯүеҫ…йҳҹеҲ—еҺҹзҗҶпјҹReentrantLock vs synchronizedпјҹ AQSдёҺCAS
    CAS зҡ„ ABA й—®йўҳеҰӮдҪ•и§ЈеҶіпјҹ AQSдёҺCAS
    Tomcat / SPI / зәҝзЁӢдёҠдёӢж–Үзұ»еҠ иҪҪеҷЁдёәд»Җд№ҲиҰҒз ҙеқҸеҸҢдәІе§”жҙҫпјҹ зұ»еҠ иҪҪжңәеҲ¶дёҺеҸҢдәІе§”жҙҫжЁЎеһӢ
    String s = new String("a") еҲӣе»әдәҶеҮ дёӘеҜ№иұЎпјҹintern() еңЁ JDK 6/7+ жңүд»Җд№ҲеҢәеҲ«пјҹ еӯ—з¬ҰдёІеә•еұӮеҺҹзҗҶдёҺStringPool
    жіӣеһӢж“ҰйҷӨдёӢзҡ„жЎҘжҺҘж–№жі•жҳҜе№Ід»Җд№Ҳзҡ„пјҹжіӣеһӢж•°з»„дёәд»Җд№ҲдёҚиғҪ new T[]пјҹ жіӣеһӢеә•еұӮеҺҹзҗҶдёҺзұ»еһӢж“ҰйҷӨ
    еҸҚе°„дёәд»Җд№Ҳж…ўпјҹMethodHandle / LambdaMetafactory еҝ«еңЁе“ӘйҮҢпјҹ еҸҚе°„дёҺMethodHandle
    select / poll / epoll зҡ„еҢәеҲ«пјҹNetty зҡ„ Reactor жЁЎеһӢжҖҺд№ҲеӣһдәӢпјҹ NIOдёҺIOжЁЎеһӢж·ұеәҰи§Јжһҗ
